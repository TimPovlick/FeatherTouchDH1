ASCOM Feather Touch Focuser Driver (C#)



ASCOM Focuser driver in C# – an in-process (assembly) based driver.



This is an ASCOM compliant driver for the original FeatherTouch® controller model DH1.

You must use the ASCOM Choser and set the Properties to match your focuser model number.

Notes:

  • This is a beta release version of the code.

  • There is no support for Temperature readings or compensation.



ASCOM Initiative



The ASCOM Initiative consists of a group of astronomy software developers and instrument vendors whose goals are to promote the driver/client model and scripting automation.
